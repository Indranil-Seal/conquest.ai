---

This file is automatically generated and updated every day at midnight UTC by the
[Update Repository Activity Stats](.github/workflows/update-stats.yml) GitHub Action.

It will populate with live data (views, clones, forks, stars) after the first workflow run.

---

## 📝 Notes on Tracking Limitations

| Activity | Username Tracking | What's Available |
|----------|-------------------|------------------|
| Opening the GitHub page | ❌ Not possible | Aggregate view counts (14-day window) |
| Clicking/viewing code files | ❌ Not possible | Not tracked by GitHub at all |
| Cloning the repository | ❌ Not possible | Aggregate clone counts (14-day window) |
| Forking the repository | ✅ Available | Usernames + timestamps via public Forks API |
| Starring the repository | ✅ Available | Usernames + timestamps via Stargazers API |

GitHub's privacy architecture (Camo image proxy, anonymous clone protocol) prevents
tracking individual usernames for page views and clones. This is by design.
